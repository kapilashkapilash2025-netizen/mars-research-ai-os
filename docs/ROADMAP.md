# Roadmap

## Phase 1 — Research foundation

- Define users, scientific scope, and trustworthy source policy
- Select public Mars datasets and APIs
- Build a cited Mars knowledge-search prototype
- Add evaluation questions and baseline quality metrics

## Phase 2 — Research workspace

- Dataset catalog and ingestion pipelines
- Notebook and visualization integrations
- Reproducible experiment tracking
- Specialized geology, atmosphere, and mission agents

## Phase 3 — AI OS

- Multi-agent research orchestration
- Simulation and planning tools
- Human review and audit trails
- Extensible plugin and tool ecosystem
